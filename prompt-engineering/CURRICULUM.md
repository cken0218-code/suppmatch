# Prompt Engineering 行銷課程大綱

## 🎯 課程目標
建立完整的 prompt engineering 知識體系，專注行銷領域 + 香港市場實戰

## 📚 課程結構

### Module 1: 基礎建立
- [ ] Context hierarchy（情境層次）
- [ ] Clarity vs Specificity 應用
- [ ] Intent alignment（意圖對齊）
- [ ] 行銷場景：廣告文案、campaign ideas、客戶 feedback 分析

### Module 2: Prompt 架構設計
- [ ] Role-Goal-Constraints-Format 框架
- [ ] 模組化結構（XML blocks、JSON schemas、paragraph frameworks）
- [ ] 不同 model 特性（Claude、ChatGPT、Gemini、Perplexity）
- [ ] 香港市場：廣東話 tone、中英夾雜、本地文化參考

### Module 3: 行銷實戰應用
- [ ] Content creation（social media、email、blog）
- [ ] Competitor research
- [ ] Customer persona building
- [ ] Campaign planning

### Module 4: Debug 和優化
- [ ] 拆解失敗原因
- [ ] 改良措辭、邏輯步驟、contextual reinforcement
- [ ] 跨 model 測試 + 一致性衡量

### Module 5: Workflow 系統設計
- [ ] 串連 prompts 成 workflows、loops、multi-step agents
- [ ] Memory layering + iterative refinement
- [ ] 長期項目管理（6 個月 campaign）

### Module 6: 文檔記錄和精通
- [ ] Meta-analysis（後設分析）
- [ ] Prompt Library 維護
- [ ] 表現評分系統（1-10）
