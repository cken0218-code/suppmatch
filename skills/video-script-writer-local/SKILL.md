# Video Script Writer Local - 视频脚本编写助手

## 功能
- 根据主题生成完整视频脚本
- 包含开场、正文、结尾
- 时间控制建议
- 多种风格支持

## 使用场景
- YouTube 视频
- 教育内容
- 产品介绍
- Vlog

---

## 核心功能

### 1. 脚本生成器

#### 输入参数
- **主题**: 视频的核心内容
- **时长**: 目标视频长度
- **风格**: 教育/娱乐/商业/Vlog
- **观众**: 目标受众
- **关键点**: 必须包含的内容点

#### 输出内容
- 完整的视频脚本
- 分段建议
- 时间分配
- 视觉建议

---

### 2. 脚本模板

#### 教育类模板
```
【开场】(0:00-0:30)
- 钩子：提出问题或痛点
- 预告：观众将学到什么
- CTA：订阅和点赞

【正文】(0:30-X:XX)
- 第一部分：概念介绍
- 第二部分：详细讲解
- 第三部分：实战案例
- 第四部分：常见错误

【结尾】(X:XX-结束)
- 总结要点
- 行动号召
- 预告下期内容
```

#### 娱乐类模板
```
【开场】(0:00-0:15)
- 有趣的开场白
- 引入主题

【正文】(0:15-X:XX)
- 主要内容
- 互动元素
- 惊喜环节

【结尾】(X:XX-结束)
- 总结
- CTA
```

---

### 3. 时间控制

#### 时间分配建议
- **5分钟视频**:
  - 开场: 30秒
  - 正文: 4分钟
  - 结尾: 30秒

- **10分钟视频**:
  - 开场: 45秒
  - 正文: 8分钟
  - 结尾: 1分钟

- **20分钟视频**:
  - 开场: 1分钟
  - 正文: 17分钟
  - 结尾: 2分钟

---

### 4. 视觉建议

#### 每个段落包含
- **场景**: 背景建议
- **道具**: 需要的道具
- **字幕**: 关键字幕提示
- **转场**: 转场效果建议

## 最佳实践

### 开场技巧
1. **前5秒最重要** - 立即抓住注意力
2. **提出问题** - 引发共鸣
3. **预告价值** - 告诉观众能得到什么
4. **CTA** - 订阅、点赞

### 正文技巧
1. **分段清晰** - 每2-3分钟一个段落
2. **视觉变化** - 每30秒变化一次画面
3. **互动元素** - 提问、投票、留言
4. **案例丰富** - 真实数据更有说服力

### 结尾技巧
1. **总结要点** - 强化记忆
2. **CTA** - 明确的行动号召
3. **预告下期** - 增加回访率
4. **感谢观看** - 建立连接
