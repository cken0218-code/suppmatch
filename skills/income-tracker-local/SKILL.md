# Income Tracker Local - 收入追踪助手

## 功能
- 多渠道收入追踪
- ROI 分析
- 预算管理
- 财务报告

## 使用场景
- 网赚业务
- YouTube 收入
- 多渠道收入
- 财务管理

## 核心功能

### 1. 多渠道收入追踪

#### 收入类型
- YouTube 广告收入
- 赞助/合作
- 产品销售
- 服务收入
- 被动收入

### 2. ROI 分析

#### 投资回报

- 工具成本 vs 收益
- 渠道对比
- 优化建议

### 3. 预算管理

#### 预算功能
- 月度预算
- 支出追踪

---

### 4. 财务报告

#### 报告类型
- 日报
- 周报
- 月报
- 年报

---

## 使用方式

### 快速追踪
```
"记录收入：
来源：YouTube
金额：$500
日期：2026-02-18"
```

### ROI 分析
```
"计算网赚 ROI：
YouTube：$500/月，10小时/周
工具成本：$50/月"
```

### 财务报告
```
"生成月度财务报告：
时间：2026年2月
收入来源：全部"
```

## 报告示例

### 月度报告
```
💰 月度财务报告
期间：2026-02-01 至 2026-02-28

💵 总收入
- YouTube：$567
- 赞助：$200

- 总计：$917 (+15%)

💸 支出
- 工具订阅：$50
- 广告投放：$100
- 总计：$150

📊 净收入：$767
ROI：512%

🏆 最佳渠道
YouTube ($567, 62%)

💡 建议
- 增加 YouTube 内容
- 减少广告支出
- 开发更多产品
```
