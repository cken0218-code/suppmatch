# Auto Publisher Local - 自动发布助手

## 功能
- 智能排程
- 多平台发布
- 批量处理
- 自动回复

## 使用场景
- YouTube automation
- 社交媒体管理
- 内容发布
- 时间优化

---

## 核心功能

### 1. 智能排程

#### 排程策略
- **最佳时间**: 基于数据分析
- **观众活跃**: 目标观众在线时间
- **避免竞争**: 避开大频道发布时间
- **间隔优化**: 视频间隔建议

---

### 2. 多平台发布

#### 支持平台
- YouTube
- B站
- TikTok
- 小红书
- Threads

#### 发布流程
1. 准备内容
2. 平台适配
3. 定时发布
4. 发布确认

---

### 3. 批量处理

#### 批量功能
- 批量上传
- 批量排程
- 批量更新
- 批量删除

---

### 4. 自动回复

#### 回复类型
- 评论回复
- 私信回复
- FAQ 自动答
- 互动增强

---

## 使用方式

### 快速排程
```
"帮我排程这个视频：
最佳时间
目标观众：美国时区"
```

### 多平台发布
```
"发布到多平台：
视频：[链接]
平台：YouTube + B站 + 小红书
时间：明天 9:00"
```

### 批量处理
```
"批量排程 5 个视频：
间隔：每2天1个
起始：明天
时间：9:00"
```
