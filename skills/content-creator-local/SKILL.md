# Content Creator Local - YouTube 内容创作助手

## 功能
- 生成吸引人的视频标题
- 编写 SEO 优化的视频描述
- 推荐相关标签和关键字
- 分析竞争对手标题

## 使用场景
- YouTube automation
- 内容创作
- 视频营销
- SEO 优化

---

## 核心功能

### 1. 标题生成器

**输入**:
- 视频主题
- 目标观众
- 风格偏好（教育/娱乐/商业）

**输出**:
- 5-10 个标题选项
- 点击率预测
- SEO 评分

**算法**:
```
1. 分析主题关键词
2. 研究热门标题模式
3. 应用心理学触发词
4. SEO 优化（关键词密度）
5. A/B 测试建议
```

**示例**:
```
输入: 主题="YouTube automation 2026", 观众="网赚者", 风格="教育"

输出:
1. "2026 YouTube Automation 完整指南：从0到月入$5000" (CTR预测: 8.5%)
2. "我如何用 YouTube Automation 30天赚$3000（附工具清单）" (CTR预测: 9.2%)
3. "YouTube Automation 2026：AI工具让你躺赚的秘密" (CTR预测: 7.8%)
...
```

### 2. 描述生成器

**输入**:
- 视频标题
- 主要内容点
- CTA（行动号召）

**输出**:
- SEO 优化的描述
- 时间戳
- 相关链接建议
- 社交媒体链接

**模板结构**:
```markdown
[开场白 - 吸引注意力]

📌 **本视频你将学到**:
• 要点1
• 要点2
• 要点3

⏰ **时间戳**:
0:00 开场
2:30 核心内容
10:45 实战案例

🔗 **相关资源**:
• 工具链接
• 参考资料

👍 **记得**:
• 订阅频道
• 点赞视频
• 留言分享你的想法

#YouTubeAutomation #网赚 #AI工具
```

### 3. 标签推荐器

**输入**:
- 视频主题
- 竞争对手标签
- 目标关键词

**输出**:
- 20-30 个标签
- 搜索量数据
- 竞争度分析

**分类**:
- **主标签** (3-5个): 高搜索量，直接相关
- **次标签** (10-15个): 中搜索量，相关话题
- **长尾标签** (10-15个): 低竞争，精准流量

### 4. 竞争分析

**输入**:
- 竞争对手视频链接
- 目标关键词

**输出**:
- 标题分析
- 描述结构
- 标签策略
- 改进建议

## 最佳实践

### 标题优化
1. **前60个字符最重要**（搜索结果显示）
2. **数字增加点击率**（"5个工具"、"2026完整指南"）
3. **情感触发词**（"秘密"、"揭秘"、"躺赚"）
4. **疑问句**（"如何..."、"为什么..."）
5. **括号补充**（"（附工具清单）"）

### 描述优化
1. **前150字符最重要**（搜索结果显示）
2. **关键词自然分布**（不要堆砌）
3. **时间戳增加观看时长**
4. **CTA 明确**（订阅、点赞、留言）
5. **链接相关资源**（增加权威性）

### 标签策略
1. **从宽到窄**（YouTube automation → AI tools → 2026 guide）
2. **混合竞争度**（高低竞争关键词都要）
3. **包含品牌**（你的频道名）
4. **使用同义词**（automation = 自动化）

## 配置选项

### 个性化设置
在 `~/.openclaw/openclaw.json` 添加:
```json
{
  "contentCreator": {
    "language": "zh-CN",
    "tone": "casual", // casual, professional, educational
    "targetPlatform": "youtube", // youtube, bilibili, tiktok
    "includeEmojis": true
  }
}
```

## 注意事项

### SEO 原则
- ✅ 关键词自然分布
- ✅ 标题吸引但诚实
- ✅ 描述提供真实价值
- ❌ 不要标题党（影响留存）
- ❌ 不要关键词堆砌

### 平台规则
- ✅ 遵守 YouTube 社区准则
- ✅ 内容真实可信
- ✅ 不误导观众
