## 🎨 自定义设置

### 调整阈值
编辑 `stock-alert.py`:
```python
# 第 105 行
buy_signals = check_buy_signals(
    rsi_threshold=35,  # RSI 阈值（默认 35）
    adx_threshold=20   # ADX 阈值（默认 20）
)
```

### 添加/移除股票
编辑 `stock-tracker.py`:
```python
# 第 18 行
DEFAULT_ASX_STOCKS = ["CBA", "BHP", "CSL", ...]
```
