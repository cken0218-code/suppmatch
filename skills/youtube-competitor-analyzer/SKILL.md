# YouTube Competitor Analyzer 🔍

> 分析竞争对手频道，搵到增长机会和差异化策略

## 功能

1. **频道分析** - 对手频道规模、内容策略
2. **视频表现** - 热门视频模式分析
3. **增长策略** - 搵到对手增长引擎
4. **差异化机会** - 搵到自己既蓝海
5. **数据追踪** - 定期监控对手动态

## 核心工作流

### Step 1: 识别竞争对手

**输入**: 你的niche/领域

**搜索方式**:
1. YouTube搜索核心关键词
2. 列出前20个频道
3. 筛选同级别对手（1k-100k订阅）

### Step 2: 频道数据收集

**关键指标**:
- 订阅数
- 总观看数
- 视频数量
- 平均观看/视频
- 频道创建时间
- 发布频率

### Step 3: 内容分析

**分析维度**:
- 内容类型 (教程/评测/Vlog)
- 视频时长分布
- 标题模式
- 标签策略
- 发布频率

### Step 4: 视频表现分析

**热门视频特征**:
- 标题关键词
- 缩图风格
- 发布时间
- 互动率 (点赞/评论/分享)

### Step 5: 增长策略挖掘

**问自己**:
- 对手点样获取首批观众？
- 咩内容表现最好？
- 有咩我可以做既差异化？

## 分析维度

### 1. 内容策略

| 维度 | 问题 |
|------|------|
| 内容类型 | 教程多定评测多？ |
| 视频时长 | 长视频定短视频？ |
| 发布频率 | 日更定週更？ |
| 系列内容 | 有无系列？ |

### 2. SEO策略

| 维度 | 问题 |
|------|------|
| 关键词 | 主攻咩关键词？ |
| 标题模式 | 有咩固定模式？ |
| 标签 | 用咩标签组合？ |

### 3. 增长引擎

| 维度 | 问题 |
|------|------|
| 社交通道 | 有无Twitter/IG？ |
| 合作 | 有无做collabs？ |
| Cross-promotion | 有无推广其他频道？ |

## 最佳实践

### 分析频率
- 重点对手: 每月1次
- 一般对手: 每季1次
- 新兴对手: 发现即分析

### 差异化策略
1. **内容差异化**: 对手做浅，我做深
2. **风格差异化**: 对手正经，我轻松
3. **格式差异化**: 对手长片，我短片
4. **领域差异化**: 搵细分市场

### 持续监控
- 记录对手发布
- 跟踪新视频表现
- 分析新趋势
