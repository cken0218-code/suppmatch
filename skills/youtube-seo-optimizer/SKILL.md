## 核心工作流

### Step 1: 关键词研究

**输入**: 视频主题/niche

**输出**:
- 主关键词 (1-2个)
- 次关键词 (3-5个)
- 长尾关键词 (5-10个)

**工具**: YouTube搜索建议、Google Trends

### Step 2: 标题优化

**原则**:
1. 包含主关键词（前面）
2. 数字+括号提升点击率
3. 情绪化词汇增加吸引力
4. 60字符以内

**示例**:
- ❌ "AI工具推荐" 
- ✅ "2026年最强的5个AI工具｜省时90%"

### Step 3: 描述优化

**结构**:
```
[第一段：钩子 + 主关键词 - 30字内]
[第二段：视频概要 - 100字]
[第三段：时间戳]
[第四段：相关链接]
[第五段：CTA]
```

### Step 4: 标签策略

**标签组合**:
1. 主关键词 (1个)
2. 次关键词 (2-3个)
3. 相关领域 (2-3个)
4. 热门话题 (1-2个)
5. 长尾变体 (2-3个)

**总数**: 15-20个标签

## 最佳实践

### 标题公式
```
[数字] + [情绪词] + [关键词] + [结果/保证]
```

示例:
- "7个免费的AI工具｜最后一个太强了"
- "5分钟学会YouTube SEO｜2026最新"

### 描述公式
```
第一行: 钩子 + 关键词
第二行: 视频价值预告
中间: 详细要点
底部: 时间戳 + CTA
```

### 标签优先级
1. 完全匹配主关键词
2. 部分匹配
3. 相关领域
4. 变体/同义词
