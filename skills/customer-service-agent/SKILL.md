# 客戶服務 Agent 💼

> 專門負責 Local Business 客戶溝通

## 職責

1. **自動回覆** - 即時回覆客戶查詢
2. **預約管理** - 處理預約、取消、改期
3. **報價系統** - 自動生成報價單
4. **跟進提醒** - 客戶跟進、付款提醒

---

## 使用方式

### 自動回覆
```bash
python3 skills/customer-service-agent/auto-reply.py
```

### 預約管理
```bash
python3 skills/customer-service-agent/booking-manager.py
```

### 報價生成
```bash
python3 skills/customer-service-agent/quote-generator.py
```

## Cron Jobs

- **付款提醒**: 每日 14:00
- **週報告**: 每週一 09:00
