## 📋 內容框架（每條片/文）

### 標準結構（5 段式）

#### 1️⃣ 開頭 Hook（15 秒內抓住注意）
**目的**: 讓觀眾停低睇落去

**技巧**:
- 問問題："你每晚瞓唔著？"
- 數據震驚："原來 70% 人都缺鎂！"
- 痛點共鳴："又係 3 點先瞓？"
- 承諾價值："今日教你 3 招即刻放鬆"

**範例**:
```
"每晚瞓喺床到眼光光？數綿羊數到 1000 都未瞓著？
今日同你分享 3 個科學方法，包你今晚好瞓！"
```

---

#### 2️⃣ 痛點/問題（30-60 秒）
**目的**: 建立共鳴，讓觀眾覺得"係我！"

**常見痛點**:
- 失眠/睡眠質素差
- 壓力大/焦慮
- 精神不振
- 身體疲勞
- 免疫力差

**範例**:
```
"你係咪都有呢啲情況？
- 瞓喺床到轉嚟轉去
- 第二朝起身好似無瞓過
- 成日覺得累
- 記憶力變差
如果你有其中兩樣，就要留意啦！"
```

---

#### 3️⃣ 解決方案（60-90 秒）
**目的**: 提供價值，建立信任

**內容類型**:
- **方法類**: 3 招改善睡眠
- **知識類**: 點解鎂可以幫你放鬆
- **產品類**: 呢隻補充劑點樣幫到你
- **工具類**: 用呢個 App 追蹤睡眠

**範例**:
```
"其實要改善睡眠，可以從三方面入手：

1. 環境調整 - 房間溫度、光線
2. 生活習慣 - 睡前 1 小時唔好睇電話
3. 營養補充 - 補充鎂、褪黑激素

今日重點講第三樣..."
```

---

#### 4️⃣ 實際示範（30-60 秒）
**目的**: 證明可行，增加說服力

**示範方式**:
- 親身體驗分享
- 用戶見證
- 科學研究數據
- Before/After 對比

**範例**:
```
"我自己試咗兩個禮拜，每晚睡前 30 分鐘食 200mg 鎂，
結果：
- 平均 15 分鐘內瞓著（以前成個鐘）
- 深層睡眠多咗 30%
- 第二朝精神好咗好多

當然，每個人體質唔同，建議先諮詢醫生..."
```

---

#### 5️⃣ 結尾 CTA（15-30 秒）
**目的**: 引導行動

**CTA 類型**:
- 訂閱/關注："記住訂閱，下集講點樣揀補充劑"
- 留言互動："你有咩睡眠問題？留言話我知"
- 點擊連結："詳細資源喺簡介欄"
- 分享："覺得有用？Share 俾朋友"

**範例**:
```
"如果你都想改善睡眠，可以睇下簡介欄嘅資源。
記住訂閱，下集會同你講點樣揀優質嘅鎂補充劑！

有咩問題？留言話我知，我會盡量答！"
```

## 📅 內容日曆規則

### 週一：精神健康 / 減壓 🧘
**主題**: 如何放鬆？精神健康？減壓

**內容方向**:
- 冥想技巧
- 呼吸練習
- 減壓方法
- 心理健康小貼士
- 工作與生活平衡

**範例標題**:
- "3 分鐘冥想，即刻放鬆"
- "點樣喺 office 減壓？"
- "焦慮發作點算？5 招急救"

---

### 週三：補充劑介紹 💊
**主題**: 各種補充劑功效、選擇、使用

**內容方向**:
- 單一補充劑深度介紹（鎂、褪黑激素、維他命）
- 補充劑組合推薦
- 點樣揀優質補充劑
- 劑量建議
- 副作用注意

**範例標題**:
- "鎂點樣幫你瞓好啲？"
- "褪黑激素 vs 鎂，邊樣好？"
- "點樣揀唔使試嘅補充劑？"

**⚠️ 注意**:
- 必須加免責聲明
- 唔好話"必定有效"
- 建議諮詢醫生

## 📝 內容範例庫

### 範例 1：補充劑介紹（週三）
**標題**: "鎂點樣幫你瞓好啲？營養師一次過話你知"

**結構**:
1. Hook: "每晚眼光光？可能係缺鎂！"
2. 痛點: 瞓唔著、肌肉緊張、焦慮
3. 解決: 鎂嘅作用、邊種鎂好
4. 示範: 個人經驗 + 研究數據
5. CTA: "想知邊隻鎂好？睇簡介欄"

---
