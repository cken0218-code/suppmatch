## 第 1 周：AI 基础 + Prompt 工程

### 学习目标
- 理解 LLM（大型语言模型）基本原理
- 掌握高效 Prompt 写法
- 学会角色设定 + Chain of Thought

### 课程内容

#### 1.1 LLM 是什么？
- GPT vs Claude vs GLM vs Gemini
- 能力与限制
- 如何选择适合嘅模型

#### 1.2 Prompt 基础公式
```
角色 + 任务 + 背景 + 输出格式
```

#### 1.3 进阶技巧
- **Few-shot Learning**：俾例子
- **Chain of Thought**：叫 AI「逐步谂」
- **角色设定**：赋予 AI 专业身份
- **约束条件**：限制长度、风格、格式

#### 1.4 实战练习
1. 写一个推销文案（Facebook / IG）
2. 写一封商业邮件
3. 总结一篇长文章
4. 分析一个产品 SWOT
5. 写一个项目提案

### 推荐资源
- OpenAI Prompt Engineering Guide
- Anthropic Claude Prompt Library
- Learn Prompting（免费课程）

- 批量生成

#### 2.3 迭代优化

#### 2.4 实战项目

## 第 3 周：数据分析 + 可视化

### 学习目标
- 用 AI 分析数据
- 生成图表 + 洞察
- 自动化报告

### 课程内容

#### 3.1 数据输入方式
- 直接贴数据
- 上传 CSV / Excel
- 连接 API

#### 3.2 分析技巧
- 描述性统计

## 第 4 周：自动化 Workflow

### 学习目标
- 理解自动化概念
- 掌握 n8n / Make.com 基础
- 整 3 个实用自动化流程

### 课程内容

#### 4.1 自动化工具比较
| 工具 | 优点 | 缺点 |
|------|------|------|
| n8n | 免费、自托管 | 需要技术背景 |
| Make.com | 易用、免费版够用 | 付费版贵 |
| Zapier | 最易用 | 免费版限制多 |

#### 4.2 常见自动化场景
- Email → Google Sheets
- 表单提交 → 通知
- 社交媒体自动发帖
- 文件自动整理

#### 4.3 实战项目
1. **自动收集**：Email 附件 → Google Drive
2. **自动通知**：Google Sheets 新行 → Telegram
3. **自动发帖**：RSS → 社交媒体

## 第 6 周：AI 图像生成

### 学习目标
- 掌握 Midjourney / DALL-E
- 写好图像 Prompt
- 商业应用

### 课程内容

#### 6.1 工具比较
| 工具 | 优点 | 缺点 |
|------|------|------|
| Midjourney | 质量最高 | 付费 |
| DALL-E 3 | 易用、in ChatGPT | 风格较统一 |
| Stable Diffusion | 免费、可自托管 | 需要技术 |

#### 6.2 图像 Prompt 技巧
```
主体 + 风格 + 光线 + 角度 + 氛围 + 尺寸
```

#### 6.3 商业应用
- 社交媒体图片
- 简报配图
- 产品 mockup
- Logo 概念

#### 6.4 实战项目
生成 10 张商业用途图片：
- 3 张社交媒体配图
- 3 张简报背景
- 2 张产品 mockup
- 2 张品牌视觉

## 第 7 周：内容创作 + 社交媒体

### 学习目标
- 制定内容策略
- 理解平台推荐机制
- 建立内容日历

### 课程内容

#### 7.1 平台特性
| 平台 | 内容类型 | 最佳频率 |
|------|----------|----------|
| Instagram | 视觉 + Reels | 每日 1-2 次 |
| TikTok | 短片 | 每日 1-3 次 |
| YouTube | 长片 + Shorts | 每周 2-3 次 |
| LinkedIn | 专业内容 | 每周 3-5 次 |

#### 7.2 推荐机制
- 完播率
- 互动率
- 内容质量
- 发布时间

#### 7.3 内容日历
- 规划主题
- 批量制作
- 定时发布

#### 7.4 实战项目
整一个「30 日内容日历」：
- 每日主题
- 内容类型
- 发布时间
- Hashtag 策略
