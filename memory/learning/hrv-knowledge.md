# HRV 健康監測知識 (2026-02-23)

## 咩係 HRV?
- Heart Rate Variability (心率變異度)
- 衡量交感神經同副交感神經既平衡

## 點樣睇？

| HRV 狀態 | 數值範圍 | 意思 |
|----------|----------|------|
| 高HRV | > 50-60 (RMSSD) | 放鬆、身體適應能力好 |
| 中HRV | 30-50 | 一般 |
| 低HRV | < 30 | 高壓力、對稱神經活躍 |

## 科學根據

### ✅ 準確既
- HRV反映自主神經系統既活動
- 高壓力時交感神經活躍 → HRV跌
- 放鬆時副交感神經主導 → HRV升

### ⚠️ 唔好完全靠
- 個人差異大（有人天生低啲）
- 要長期數據先準（一時既唔作實）
- 運動、咖啡、睡眠都會影響
- 唔係診斷工具，只係參考指標

## 對身體既影響

- **長期低HRV** → 心血管問題、焦慮、壓力大既關聯
- **長期高HRV** → 身體適應能力較好

## 點樣改善 HRV？

1. **深呼吸 / 冥想**
2. **充足睡眠**
3. **適量運動**
4. **減少咖啡因**
