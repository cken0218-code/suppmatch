**时间**: 2026-02-18 14:35
**错误信息**: `HTTP 400: invalid params, unknown model 'm2.5' (2013)`

**原因**:
- 配置使用了错误的模型 ID: `m2.5`
- 正确的模型 ID 应该是: `MiniMax-M2.5`

**修复方案**:
```json

"id": "m2.5"

// 正确配置
"id": "MiniMax-M2.5"
```

**修复状态**: ✅ 已修复
**文件**: `~/.openclaw/openclaw.json`

**教训**:
- 使用官方文档确认模型 ID
- 不要猜测 API 参数
