### 1. HTTP 401 Authentication Error (09:00)
**错误信息**: `HTTP 401 authentication_error: invalid api key`
**影响**: Cron 任务失败
**原因**: 某个 API key 无效或过期
**解决**: 检查 `~/.openclaw/openclaw.json` 中的 API keys

### 3. Message Send 缺少 Target (14:02, 15:03, 17:01, 18:07)
**错误信息**: `Action send requires a target`
**影响**: Discord/Telegram 通知失败
**原因**: 某些 cron 任务调用 `message` 时未指定 `target`
**解决**: 检查 cron 任务配置，确保所有 message 调用都有 target

---

### 4. Telegram Chat ID 解析失败 (18:08)
**错误信息**: `Telegram recipient @default could not be resolved to a numeric chat ID`
**影响**: Telegram 通知失败
**原因**: `@default` 不是一个有效的 Telegram chat ID
**解决**: 
- 需要先与 Telegram bot 对话
- 获取正确的 chat ID（数字格式）

---

### 5. Web Fetch 403 错误 (19:00)
**错误信息**: `Web fetch failed (403)`
**原因**: Cloudflare 或其他防护拦截
**解决**: 改用 DuckDuckGo 搜索或 browser automation
