#### 2. Web Fetch 403 Errors (07:10-07:11)
**错误信息**：
```
[tools] web_fetch failed: Web fetch failed (403)
- Cloudflare blocks
- "Attention Required! | Cloudflare"
```

**原因**：多个网站有 Cloudflare 保护，直接 fetch 被 block

**受影响网站**：
- CoinGecko
- CoinDesk
- theblock.co
- 其他加密货币网站

**解决方案**：
- ✅ 改用 DuckDuckGo Lite 搜索
- ✅ 用 browser automation 代替 web_fetch

**状态**：✅ 已有 fallback

---

### 💡 学到的教训

1. **Discord messaging**：
   - 永远用 `user:ID` 格式
   - 唔好靠 username

2. **Web scraping**：
   - Cloudflare 保护嘅网站要避开
   - DuckDuckGo 系可靠 fallback

3. **File operations**：
   - Read 前先 check 文件是否存在
   - 唔好尝试 read 目录
