# Gamification 系统设计

**参考**: cheyuwu345 既 Muse RPG 系统

---

## 🎮 Octalysis 框架

基于八角分析法 (Yu-kai Chou)：

| 角落 | 名称 | 应用 |
|------|------|------|
| 1 | 史诗意义 & 召唤 | 长期目标、AI 使命 |
| 2 | 发展 & 成就感 | 技能升级、成就徽章 |
| 3 | 创意拥有感 | 自定义选项 |
| 4 | 所有权 & 拥有感 | 收集、解锁 |
| 5 | 社交影响 | 排行榜、分享 |
| 6 | 稀缺感 | 限时任务 |
| 7 | 未知感 | 随机事件、Mystery Box |
| 8 | 损失避免 | Streak 保护 |

## 🏆 成就系统

### 成就徽章
| 成就 | 条件 | 奖励 |
|------|------|------|
| 🌅 早起鸟 | 08:00 前 first heartbeat | +10 XP |
| 📚 学识渊博 | 完成 10 次 research | +50 XP |
| 💰 股神 | 连续 5 次 signal 正确 | +100 XP |
| 📺 YouTube 大师 | 产出 100 条 content ideas | +100 XP |
| 🛡️ 安全卫士 | 扫描 50 个 skills | +50 XP |
| 🎯 任务完成 | 完成 100 个 tasks | +200 XP |
| 🔥 Streak 7 | 连续 7 天活跃 | +100 XP |
| 🔥 Streak 30 | 连续 30 天活跃 | +500 XP |

## 🎲 随机事件

### Mystery Box (每週)
- 小幅强化包 (20%): +10 XP
- 中幅强化包 (10%): +50 XP
- 大幅强化包 (5%): +100 XP
- 随机技能点 (5%): 随机 +1 skill

### 随机事件
- 灵感爆发: 触发创意任务
- 技术突破: 触发学习任务
- 市场波动: 触发股票分析

## 🔧 实现方式

### 1. Heartbeat 时检查
- 每日任务完成度
- XP 累加
- Streak 检查

### 2. Cron Job 週报
- 每週六总结
- 发放 Mystery Box
- 更新排行榜

---

## ⚠️ 注意事项

- XP 不要太易攞（会冇动力）
- Streak 断咗要可以补（1次/月）
- 神秘感 - 唔好全部公布
