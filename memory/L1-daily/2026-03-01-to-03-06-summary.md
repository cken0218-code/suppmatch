## 🛠️ 3月3号：技术调试

### API Key 认证问题
- **问题**：Sub-agents / Cron jobs 无 API key
- **原因**：Isolated sessions 搵唔到 auth profiles
- **解决**：Copy auth profiles 到 subagent 目录
  ```bash
  cp ~/.openclaw/agents/main/agent/auth-profiles.json ~/.openclaw/agents/main/subagent/
  ```
