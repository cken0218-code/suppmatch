### 3. Agentic Engineering Patterns 实践指南 🛠️

**核心模式 (Simon Willison):**

**Principles:**
- **Writing code is cheap now** - 代码成本极低，大胆尝试
- **Hoard things you know how to do** - 积累可复用的模式

**Testing and QA:**
- **Red/green TDD** - 测试驱动开发
- **First run the tests** - 先跑测试再写代码

**Understanding code:**
- **Linear walkthroughs** - 线性代码演示
- **Interactive explanations** - 交互式解释

**Annotated prompts:**
- 保存成功的 prompt 模板
- 记录失败案例
- 建立最佳实践库

**对 OpenClaw 的启示:**
