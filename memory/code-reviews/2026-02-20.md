## [autonomous-runner] Review
- 路徑: `skills/autonomous-runner/autonomous-runner.sh`
- 狀態: ⚠️ 需要注意
- 發現:
  - **命令注入風險**：`--command` 參數直接傳遞給 shell 執行（第97行）：
    ```bash
    nohup bash -c "$COMMAND" > "... 2>&1 &
    ```
  - 如果 COMMAND 包含特殊字符，可能導致命令注入
  - `LABEL` 和 `TASK` 參數也直接使用在路徑和文件名中，缺少消毒
- 建議:
  - 對 COMMAND 參數進行基本驗證
  - 使用 `"$COMMAND"` 確保正確引用
  - 避免在命令中使用未經處理的用戶輸入
